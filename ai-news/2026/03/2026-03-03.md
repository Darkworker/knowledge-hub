# AI Agent 领域新闻摘要 - 2026年3月3日

## 一、重要产品/工具发布

### 1. OpenAI Operator & Deep Research
- OpenAI 发布首款 AI Agent 产品 **Operator**，可自动执行浏览器操作完成订餐、购物、订票等任务
- **Deep Research** 功能能在 5-30 分钟内生成专业领域深度研究报告，由 o3 模型支持

### 2. 微软 Azure AI Foundry Agent Service
- 正式支持多 Agent 编排，集成 Semantic Kernel 和 AutoGen
- 支持 MCP 协议和 A2A 协议，推出 Agent 可观测性仪表盘

### 3. 谷歌 Gemini 2.0 + 三大 Agent 原型
- **Project Astra**：通用多模态助手
- **Project Mariner**：浏览器助手
- **Jules**：GitHub 集成编程助手

### 4. 阿里云 Qwen2.5-Max & Qwen2.5-VL
- 超大规模 MoE 模型，支持 Computer Use 场景
- 可直接作为视觉 Agent 操作计算机和手机完成复杂任务
